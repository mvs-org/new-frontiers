EVM Compatibility

Metaverse allows you to run EVM compatible contracts. Ethereum compatible RPC methods improve dapp compatibility.

Digital Identity

Metaverse Avatars are digital identities protected by the Metaverse hybrid consensus.

Digital Assets

Fungible as well as non-fungible assets can easily be created, transferred and traded.

The MetaverseVM is not a bridge to another chain and is not a second layer Ethereum chain. The Metaverse VM enables you to run dapps natively, but does not deal with the issues of communicating with other Ethereum-based networks. It is the an independent ledger that provides an EVM compatible Virtual Machine for the Metaverse Blockchain project.
